# Classification des Fleurs d'Iris
Le jeu de données Iris est un jeu classique en apprentissage automatique.
il contient 150 échantillons de fleurs appartenant a trois especes :
iris setosa
iris versicolor
iris virginica
Chaque fleur est décrite par quatre caractéristiques :
Longueur du sépale (cm) 
Largeur du sépale (cm) 
Longueur du pétale (cm) 
Largeur du pétale (cm)
